28. Consider three variables $a, $b, $c, apply var_dump() function to evaluate if the values
are equal or not.
answer
<?php
$a = 10;
$b = "10";
$c = 10;

var_dump($a == $b); // true
var_dump($a === $b); // false
var_dump($a === $c); // true
?>
output
bool(true)
bool(false)
bool(true)
