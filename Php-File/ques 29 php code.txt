29. Infer the usage of the following PHP functions:
(i) rand() (ii) abs() (iii) strreplace() (iv) pi( ) (v) ceil()
answer
<?php
echo rand(1, 100) . "<br>";   // random number
echo abs(-50) . "<br>";       // absolute value
echo str_replace("world", "PHP", "Hello world") . "<br>"; // replace
echo pi() . "<br>";           // Pi value
echo ceil(4.3) . "<br>";      // round up
?>
